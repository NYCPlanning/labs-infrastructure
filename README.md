# NYC Planning Labs Infrastructure

## Usage

1. Install dependencies.
    * Python 3
    * [Pipenv](https://docs.pipenv.org)
    * [Vagrant](https://www.vagrantup.com/)
    * [Virtualbox](https://www.virtualbox.org/)
1. Run the setup:

    ```shell
    pipenv install
    pipenv shell

    vagrant up
    vagrant provision
    ```
